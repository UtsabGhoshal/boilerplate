## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v16 or newer
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
git clone https://github.com/UtsabGhoshal/boilerplate.git
cd boilerplate
npm install
```

### Running the Project

```bash
npm start
```
